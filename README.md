# AXON.AI 🤖

AXON is a personal AI assistant designed to help with everyday tasks, productivity, learning, and automation.  
It is built to be customizable and expandable, allowing users to adapt it to their own workflow.

---

## 🚀 Features

- 💬 Chat-based AI assistant
- 🧠 Context-aware responses (memory system planned)
- 🔧 Task automation tools
- 📁 File handling and note management
- 🌐 Web information access (planned)
- 🧩 Modular plugin system (planned)
- ⚡ Fast and lightweight architecture

---

## 🎯 Goal

The goal of AXON is to create a **personal AI system** that is:
- More adaptable to individual users
- Expandable with custom tools (only pre-created plugins)
- Useful for coding, studying, and productivity

---

## 🛠️ Tech Stack

- Python 🐍
- FastAPI (backend API)
